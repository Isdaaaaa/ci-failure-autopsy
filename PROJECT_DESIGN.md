# Visual Identity

## Personality
Confident, forensic, and calm. Feels like a trustworthy investigator who quickly isolates the signal from noisy logs.

## Colors
- Navy (#0F172A) background for terminal feel
- Cyan (#06B6D4) for highlights and links
- Amber (#F59E0B) for warnings and culprit markers
- Slate (#E2E8F0) for text and dividers

## Typography
- Heading: Inter SemiBold
- Body: Inter Regular/Medium
- Monospace snippets: JetBrains Mono

## Components
- Paste/upload panel with drag-and-drop
- Timeline with step badges and evidence excerpts
- Signature cards showing error type, confidence, and cited lines
- Fix checklist with checkboxes and quick-copy buttons
- PR plan box with copy-to-clipboard actions

## Layout
Two-column desktop layout: left for log input and timeline; right for signatures, fixes, and PR plan. Mobile stacks vertically with sticky top actions.

## Inspiration
- Datadog incident timelines
- Linear issue cards for concise detail
- Terminal-inspired log viewers with syntax highlighting
