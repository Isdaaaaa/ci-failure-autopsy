# Roadmap

## Phase 1 — Bootstrap
Set up repo, Next.js app shell, Tailwind, lint/test tooling, and basic layout.

## Phase 2 — Ingestion & Parsing
Implement paste/upload form, detect log format, extract sections, and store runs.

## Phase 3 — Classification & Timeline
Classify failure stage, pull key error signatures, and render a chronological timeline with evidence snippets.

## Phase 4 — Suggestions & PR Plan
Generate fix checklist, probable causes with confidence notes, and a suggested commit message/PR description using extracted evidence.

## Phase 5 — Demo Data & Polish
Seed sample logs, add screenshots/GIF, tighten copy, and prep README for showcase.
